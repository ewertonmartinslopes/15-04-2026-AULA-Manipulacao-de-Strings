# 🐍 Python – Manipulação de Strings

Este repositório reúne atividades desenvolvidas no **Google Colab** com foco na manipulação de **strings** utilizando Python.

Os exercícios apresentam técnicas utilizadas no tratamento de dados textuais, simulando situações comuns em sistemas de cadastro, como limpeza de informações, padronização de textos, substituição de caracteres e extração de partes específicas de uma string.

---

## 🎯 Objetivos do Projeto

* Aprender a manipular textos utilizando Python.
* Praticar os principais métodos da classe `string`.
* Desenvolver habilidades para tratamento e padronização de dados.
* Aplicar operações de indexação e fatiamento de textos.
* Compreender como preparar informações para utilização em sistemas.

---

## 📚 Atividades Desenvolvidas

### 🔹 Atividade 1 – Tratamento de Nome e E-mail

Programa responsável por realizar diversas operações em um nome e endereço de e-mail.

As funcionalidades implementadas incluem:

* Remoção de espaços em branco utilizando `strip()`.
* Conversão de caracteres para letras minúsculas com `lower()`.
* Substituição do domínio do e-mail utilizando `replace()`.
* Contagem da quantidade de caracteres com `len()`.
* Identificação da primeira e da última letra utilizando indexação.

**Conceitos praticados:**

* Manipulação de strings
* Métodos `strip()`
* Método `lower()`
* Método `replace()`
* Função `len()`
* Indexação de strings

---

### 🔹 Atividade 2 – Cadastro e Padronização de Produto

Programa que solicita ao usuário informações sobre um produto, categoria e código, realizando o tratamento dos dados antes da exibição.

As operações realizadas incluem:

* Remoção de espaços extras.
* Conversão para letras minúsculas.
* Conversão do código para letras maiúsculas.
* Substituição de caracteres utilizando `replace()`.
* Contagem de caracteres do nome do produto.
* Extração da primeira letra do produto.
* Extração dos quatro primeiros caracteres do código.

**Conceitos praticados:**

* Entrada de dados (`input`)
* Manipulação de strings
* Padronização de textos
* Indexação
* Fatiamento (`slice`)
* Tratamento de informações
